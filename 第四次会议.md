###### 1. 用户注册

**接口说明**: 创建新用户账号

- **URL**: `/user/register/`
- **方法**: `POST`
- **权限**: 无需认证

**请求参数**:
| 字段名 | 类型 | 必需 | 描述 | 示例 |
|--------|------|------|------|------|
| username | string | 是 | 用户名(3-20字符) | "john" |
| email | string | 是 | 邮箱地址 | "john@example.com" |
| password | string | 是 | 密码(最少8位) | "password123" |
| confirm_password | string | 是 | 确认密码 | "password123" |
